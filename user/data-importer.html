Data Importer

The data importer is used to import already existing data to the spacialist software using CSV files.
As there are many different non-trivial Attribute Types supported, which require the data to be formatted properly Visit the Attribute Types page to get an overview of all attribute types and their respective import format.

If you want to have an example of the file structure, you may Export a an example entity to get a scaffold for the import.
