# I18N - Internationalisation

Plugins may use the vocabulary that is provided by the core, using vue's 
i18n feature. This can be accomplished by extracting the `t` function from SpPS.

```js
    const t = SpPS.data.t;
    
    // Currenlty (v12) t is also available directly on the window object
    const t = window.t;
```

## Core Messages

Core messages can be called regulary:

```js
    t('global.cancel'); // -> 'Cancel'
```

## Plugin Messages

Plugin messages get registered under the plugin path:

```json title="en.json"
{
    "pluginName": {
        "celebration": "🎉 Congratulations!"
    }
}
```

Will be located at 

```js
    t('plugin.pluginName.celebration'); // -> "🎉 Congratulations!"
```
