Overview

Spacialist is a versatile, highly customizable, user-friendly database management solution for the sciences and the digital humanities in peticular. The application has a wide variety of features.

Web-based Software

The client can be accessed on any computer using a standard browser.

Customizable Data Model

Highly flexible project-specific data models with a wide range of attribute types.

Thesaurus

Multilingual and centrally controlled vocabulary for compliant with the SKOS standard.

Bibliography

Basic bibliography management, allowing BibTex import and export.

User & Role Management

Manages user accounts and assign them specific roles with fully customizable rights and permissions.

Collaboration

Working on the same dataset as a team, managing certainty, literature and commenting on the data directly.

Plugins

Application is extensible through a plugin system. Adding existing functionality or writing custom plugins.

Files

The file plugin enables a file system, where files can be linked to single entities.

Map

The map plugin provides a light-weight GIS interface to visualize the data on maps.

Analysis

User-Interface to run elaborate queries without the need to write SQL.
